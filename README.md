# Pre Interview Form from PRIMO

For implementing function with this interface
`merge (int[] collection_1, int[] collection_2, int[] collection_3) : int []`
That returns sorted array by ascending.

given

- `collection_1`, `collection_3` already sorted from min(0) to max
- `collection_2` already sorted from max to min(0)

> Note: `merge` function does not verify given collections and assume they follow the instructions

## Setup

```bash
npm i
```

## Tests

```bash
npm run test
```

## Usage

```bash
npm run dev
```

Manually input collection data separated by comma (`,`)

```
collection_1: 1,2,3
collection_2: 9,8,7
collection_3: 4,5,6
```
